# Default fonts list {#fonts-default .reference}

The default list of fonts used by the Tiny editors' font picker.

```
  fonts: [
    'Arial',
    'Comic Sans MS',
    'Consolas',
    'Courier',
    'Georgia',
    'Impact',
    'Tahoma',
    'Times New Roman',
    'Trebuchet MS',
    'Verdana'
  ],
```
